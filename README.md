# Instructions

1. ติดตั้ง Node.js และ Express
   ```bash
   npm install express sqlite3

2. run application
   ```bash
    nodemon server.js

4. ทดสอบ API
    - POST Request ที่ http://localhost:3000/add_product เพื่อเพิ่มสินค้า

    - GET Request ที่ http://localhost:3000/get_products เพื่อดึงรายการสินค้า
